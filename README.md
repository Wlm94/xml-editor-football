🚀 XML Editor Football  

**Un éditeur XML pour les données football en ligne !**  

✨ Fonctionnalités  
- Charger/modifier des fichiers XML football  
- Visualisation arborescente  
- Export en JSON/CSV  

🛠️ Installation  
```bash
git clone https://github.com/Wlm94/xml-editor-football
cd ton-projet
ouvrir index.html  
```
