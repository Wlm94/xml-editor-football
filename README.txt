Projet effectué à l'aide de VSCODE et de WAMPSERVER. 
Utilisation de HTML, CSS, JAVASCRIPT et PHP. 

L'exécution du fichier XML met quelques secondes avant son apparition, pendant et après la modification du fichier sur l'interface WEB. Si le fichier est modifié, il ne faut pas oublier d'appuyer sur le bouton Enregistrer, un petit message de pop-up apparaîtra alors sur l'éditeur XML.

La barre de recherche permet de rechercher un élément rapidement. 

Les boutons de navigations précédent et suivant, permettent la navigation entre les éléments du fichier XML.

La case juste en dessous des deux boutons indique le nombre d'éléments, mais également si aucun élément n'a été trouvé.
